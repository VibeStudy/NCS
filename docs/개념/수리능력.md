## 기초통계능력

### 통계의 개념

| 용어 | 설명 |
|------|------|
| **통계** | 집단 현상을 수량으로 표현하고 분석하는 방법 |
| **모집단** | 통계 조사 대상이 되는 전체 집단 |
| **표본** | 모집단에서 실제로 추출하여 조사한 일부 집단 |
| **전수조사** | 모집단 전체를 조사하는 방법 (정확하지만 비용·시간 많이 소요) |
| **표본조사** | 모집단의 일부(표본)를 추출하여 조사하는 방법 (효율적이나 오차 발생 가능) |
| **변수** | 조사 대상에서 측정되는 특성·항목 |
| **자료** | 변수를 측정하여 얻은 값의 집합 |

**통계의 종류**

| 종류 | 설명 | 예시 |
|------|------|------|
| **정태통계** | 특정 시점의 상태를 나타내는 통계 | 인구 수, 재고량, 자산 현황 |
| **동태통계** | 일정 기간 동안의 변화·흐름을 나타내는 통계 | 출생률, 무역량, 생산량 변화 |

**자료의 종류**

| 종류 | 설명 | 예시 |
|------|------|------|
| **질적 자료 (범주형)** | 수치로 표현할 수 없는 속성·범주 | 성별, 혈액형, 직업 |
| **양적 자료 (수치형)** | 수치로 측정 가능한 자료 | 키, 몸무게, 점수 |
| **이산형 자료** | 셀 수 있는 정수값 | 학생 수, 불량품 수 |
| **연속형 자료** | 연속적인 실수값 | 온도, 길이, 시간 |

---

### 대표값

자료 전체의 특징을 하나의 수로 나타낸 값

| 종류 | 설명 | 공식 |
|------|------|------|
| **평균** | 자료의 합을 개수로 나눈 값 | (자료의 총합) ÷ (자료의 개수) |
| **중앙값 (중위수)** | 자료를 크기순으로 나열했을 때 가운데 위치하는 값 | 홀수 개: 가운데 값 / 짝수 개: 가운데 두 값의 평균 |
| **최빈값** | 자료 중 가장 자주 나타나는 값 | 빈도수가 가장 높은 값 (여러 개일 수 있음) |

> 극단값(이상치)이 있을 때: 평균보다 **중앙값**이 더 대표성 있음

---

### 산포도

자료가 대표값을 중심으로 얼마나 흩어져 있는지를 나타내는 값

| 종류 | 설명 | 공식 |
|------|------|------|
| **범위** | 최댓값 - 최솟값 | 최댓값 - 최솟값 |
| **분산** | 편차 제곱의 평균 | (각 편차²의 합) ÷ (자료의 개수) |
| **표준편차** | 분산의 양의 제곱근. 단위가 원래 자료와 같음 | √분산 |
| **편차** | 각 자료값 - 평균 | (자료값) - (평균) |

> 편차의 합은 항상 **0**
> 분산·표준편차가 클수록 자료가 더 **넓게 퍼져** 있음

---

### 도수분포표

| 용어 | 설명 |
|------|------|
| **계급** | 자료를 나눈 구간 (예: 60이상 70미만) |
| **계급의 크기** | 각 구간의 너비 (= 상한 - 하한) |
| **계급값** | 계급의 중앙값 = (하한 + 상한) ÷ 2 |
| **도수** | 각 계급에 속하는 자료의 개수 |
| **상대도수** | 도수 ÷ 전체 도수 (비율, 합계 = 1) |
| **누적도수** | 첫 계급부터 해당 계급까지의 도수 합계 |

---

### 확률

| 개념 | 설명 | 공식 |
|------|------|------|
| **확률** | 어떤 사건이 일어날 가능성 | (사건 A의 경우의 수) ÷ (전체 경우의 수) |
| **여사건** | 사건 A가 일어나지 않을 사건 | P(Aᶜ) = 1 - P(A) |
| **합사건** | A 또는 B가 일어날 사건 | P(A∪B) = P(A) + P(B) - P(A∩B) |
| **배반사건** | 동시에 일어날 수 없는 두 사건 | P(A∩B) = 0 → P(A∪B) = P(A) + P(B) |
| **독립사건** | 두 사건이 서로 영향을 주지 않음 | P(A∩B) = P(A) × P(B) |

> 확률의 범위: **0 ≤ P(A) ≤ 1**

---

### 경우의 수

| 법칙 | 설명 | 예시 |
|------|------|------|
| **합의 법칙** | 두 사건이 동시에 일어나지 않을 때, 경우의 수의 합 | A 또는 B = m + n |
| **곱의 법칙** | 두 사건이 동시에 또는 연이어 일어날 때, 경우의 수의 곱 | A 그리고 B = m × n |
| **순열 (P)** | n개 중 r개를 순서 있게 나열 | nPr = n! ÷ (n-r)! |
| **조합 (C)** | n개 중 r개를 순서 없이 선택 | nCr = n! ÷ (r! × (n-r)!) |

> 순열: 순서 O / 조합: 순서 X

## 도표분석능력

### 도표의 목적

복잡한 수치 자료를 시각적으로 표현하여 정보를 빠르고 정확하게 전달하는 것

---

### 도표의 용도

| 용도 | 설명 | 주로 쓰이는 도표 |
|------|------|-----------------|
| **경과** | 시간의 흐름에 따른 변화·추이를 나타냄 | 꺾은선그래프, 층별그래프 |
| **내역** | 전체를 구성하는 각 항목의 내용·비율을 나타냄 | 원그래프, 띠그래프, 막대그래프 |
| **비교** | 항목 간·집단 간 수치를 나란히 비교 | 막대그래프, 방사형그래프 |
| **분포** | 자료가 어떻게 퍼져 있는지 나타냄 | 히스토그램, 도수분포표 |
| **상관** | 두 변수 간의 관계·연관성을 나타냄 | 점그래프(산점도) |
| **계산** | 도표를 이용해 직접 수치를 산출하거나 계산 결과를 확인 | 계산도표, 환산표 |

---

### 도표의 종류

| 도표 종류 | 특징 | 적합한 상황 |
|-----------|------|-------------|
| **막대그래프** | 항목별 수량을 막대 길이로 비교 | 항목 간 크기 비교 |
| **꺾은선그래프** | 시간에 따른 변화를 선으로 연결 | 시간적 변화·추세 파악 |
| **원그래프** | 전체에서 각 항목의 비율을 부채꼴로 표시 | 구성 비율·점유율 비교 |
| **띠그래프** | 전체를 100% 기준으로 각 항목의 비율을 가로 띠로 표시 | 여러 집단 간 비율 비교 |
| **층별그래프** | 각 항목의 실제 수치를 쌓아 올려 표현 | 전체 합계 + 구성 변화 동시 파악 |
| **점그래프 (산점도)** | 두 변수 간의 관계를 점으로 표시 | 상관관계 파악 |
| **방사형그래프 (레이더)** | 여러 항목을 방사형으로 나타내어 균형 파악 | 다항목 능력·균형 비교 |
| **히스토그램** | 연속 자료를 계급별로 막대로 표시 (막대 사이 간격 없음) | 도수분포 표현 |

> 띠그래프: **비율(%)** 기준 / 층별그래프: **실제 수치(절댓값)** 기준

![도표 종류 예시](images/graph.png)

### 도표분석의 유의사항 

- 자신의 업무와 관련된 지식의 상식화 
- 도표에 제시된 자료의 의미에 대한 숙지 
- 도표로부터 파악 가능한 것과 불가능한 것 구별 
- 총량의 증가와 비율의 증가 구별 
- 백분위수와 사분위수 의미 이해

### 백분위수와 사분위수

**백분위수 (Percentile)**

자료를 크기 순서로 나열했을 때 하위 n%에 해당하는 값

| 표기 | 의미 | 예시 |
|------|------|------|
| P25 | 하위 25%에 해당하는 값 | 시험 점수 P25 = 60점 → 하위 25% |
| P50 | 하위 50%에 해당하는 값 (= 중앙값) | |
| P75 | 하위 75%에 해당하는 값 | |
| P90 | 하위 90%에 해당하는 값 | |

> "백분위수 80"이면 → 전체의 80%가 그 값보다 낮다는 의미

---

**사분위수 (Quartile)**

자료를 크기 순서로 나열한 뒤 4등분하는 3개의 값

| 사분위수 | 표기 | 의미 |
|----------|------|------|
| **제1사분위수** | Q1 | 하위 25% 위치의 값 (= P25) |
| **제2사분위수** | Q2 | 하위 50% 위치의 값 (= 중앙값, P50) |
| **제3사분위수** | Q3 | 하위 75% 위치의 값 (= P75) |

> **사분위 범위(IQR)** = Q3 - Q1 → 중간 50% 자료의 퍼짐 정도를 나타냄
> IQR이 작을수록 자료가 중앙에 집중되어 있음

## 도표작성능력

### 도표작성의 이점

| 이점 | 설명 |
|------|------|
| **시각적 명확성** | 복잡한 수치 자료를 한눈에 파악할 수 있음 |
| **비교 용이** | 항목 간·시기 간 차이를 직관적으로 비교할 수 있음 |
| **설득력 강화** | 숫자보다 그래프가 주장의 근거로 더 효과적으로 전달됨 |
| **정보 압축** | 방대한 데이터를 간결하게 요약하여 전달 가능 |
| **의사결정 지원** | 추세와 패턴을 빠르게 파악하여 판단에 활용 |

---

### 도표 작성의 절차

1. 도표 종류 결정
2. 가로축과 세로축의 항목 결정
3. 가로축과 세로축의 눈금 결정 
4. 자료를 가로축과 세로축이 만나는 위치에 표시 
5. 표시된 점에 따라 도표 작성
6. 도표의 제목 및 단위 표시

---

### 도표작성 시 유의사항

**전반적인 유의사항**

1. 보기 쉽게
2. 단순한 내용 
3. 순서가 없는 것은 큰 것부터, 왼쪽에서 오른쪽으로, 위에서 아래로
4. 적정한 눈금 설정 
5. 수치 생략 없이 
6. 컴퓨터 전산 그래프 이용

**그래프별 유의사항**

| 그래프 | 유의사항 |
|--------|----------|
| **선(꺾은선) 그래프** | 1. 세로축에 수량, 가로축에 명칭 <br> 2. L자 모양의 축 사용 <br> 3. 세로축의 눈금을 가로축의 눈금보다 크게 <br> 4. 선의 종류가 두 가지 이상인 경우 선마다 명칭 기입 <br> 5. 중요한 선은 굵게 하거나 색을 다르게 |
| **막대그래프** | 1. 일반적으로 세로 그래프 사용 <br> 2. L자 모양의 축 사용, 가로 막대는 사방을 틀로 감쌈 <br> 3. 가로축은 명칭, 세로축은 수량 <br> 4. 막대 수가 많을 경우 눈금선 기입 <br> 5. 막대의 폭은 모두 같게 |
| **원그래프** | 1. 정각 12시 선이 시작선, 시계 방향으로 작성 <br> 2. 구성 비율이 큰 순으로 표시, 기타는 마지막에 <br> 3. 항목의 명칭은 같은 방향으로 기록, 기록이 힘든 경우 지시선을 써서 기록 |
| **층별그래프** | 1. 층의 세로/가로 표현은 작성자의 기호에 따라 판단 <br> 2. 눈금은 다른 그래프보다 적게 표시 <br> 3. 눈금선은 제외, 층별로 색이나 모양을 다르게 표현 <br> 4. 같은 항목은 옆의 층과 선으로 연결해 보기 쉽게 표현 <br> 5. 중요 항목은 위에서 아래로, 왼쪽에서 오른쪽으로 |

---

### 도표작성의 실제

엑실 프로그램을 활용한 도표작성의 단계
1. 자료의 입력
2. 삽입-그래프의 종류 선택 
3. 데이터의 범위와 계열 수정
4. 범례 수정

## 문제 풀이 수식

### 거리·속력·시간

| 구하는 것 | 공식 |
|-----------|------|
| 거리 | 거리 = 속력 × 시간 |
| 속력 | 속력 = 거리 ÷ 시간 |
| 시간 | 시간 = 거리 ÷ 속력 |

> 단위 통일 주의 (km/h ↔ m/s: ÷ 3.6)

---

### 작업량

| 구하는 것 | 공식 |
|-----------|------|
| 작업량 | 작업량 = 작업속도 × 시간 |
| 작업속도 | 작업속도 = 작업량 ÷ 시간 |
| 시간 | 시간 = 작업량 ÷ 작업속도 |

> 전체 작업량을 **1**로 놓고 풀기
> 여러 명이 함께 할 때: 각자의 하루 작업량을 합산

---

### 농도

| 구하는 것 | 공식 |
|-----------|------|
| 농도(%) | 농도 = (용질의 양 ÷ 용액의 양) × 100 |
| 용질의 양 | 용질 = 용액 × 농도 ÷ 100 |
| 용액의 양 | 용액 = 용질 ÷ 농도 × 100 |

> 용액 = 용질 + 용매
> 물을 섞으면 용질은 그대로, 용액만 증가

---

### 정가·원가·이익·할인율

| 용어 | 공식 |
|------|------|
| 정가 | 정가 = 원가 × (1 + 이익률) |
| 판매가 | 판매가 = 정가 × (1 - 할인율) |
| 이익 | 이익 = 판매가 - 원가 |
| 이익률 | 이익률 = 이익 ÷ 원가 × 100 |
| 할인율 | 할인율 = 할인액 ÷ 정가 × 100 |

---

### 증감률·변화량·비율·비중

| 구하는 것 | 공식 |
|-----------|------|
| 증감률(%) | (나중값 - 처음값) ÷ 처음값 × 100 |
| 변화량 | 나중값 - 처음값 |
| 비율(%) | 부분 ÷ 전체 × 100 |
| 비중(%) | 특정 항목 ÷ 전체 합계 × 100 |
| 평균 | 합계 ÷ 개수 |

> 증감률: 변화의 **상대적 크기** (%) / 변화량: 변화의 **절대적 크기**
> 비율과 비중은 같은 공식이나, 비중은 구성 요소 간 상대적 중요도를 강조할 때 사용

---

### 경우의 수

| 종류 | 공식 | 특징 |
|------|------|------|
| **합의 법칙** | m + n | A 또는 B (동시에 일어나지 않음) |
| **곱의 법칙** | m × n | A 그리고 B (연이어 일어남) |
| **순열 nPr** | n! ÷ (n-r)! | n개 중 r개를 **순서 있게** 나열 |
| **조합 nCr** | n! ÷ (r! × (n-r)!) | n개 중 r개를 **순서 없이** 선택 |
| **중복순열** | nᵣ | n개 중 r개를 순서 있게, 중복 허용 |
| **원순열** | (n-1)! | n개를 원형으로 배열 |

> 순열: 순서 O / 조합: 순서 X
> n! = n × (n-1) × (n-2) × … × 1

### 확률

| 종류 | 공식 |
|------|------|
| **기본 확률** | P(A) = 사건 A의 경우의 수 ÷ 전체 경우의 수 |
| **여사건** | P(Aᶜ) = 1 - P(A) |
| **합사건 (일반)** | P(A∪B) = P(A) + P(B) - P(A∩B) |
| **합사건 (배반)** | P(A∪B) = P(A) + P(B) |
| **독립사건의 곱** | P(A∩B) = P(A) × P(B) |
| **조건부 확률** | P(A\|B) = P(A∩B) ÷ P(B) |

> 확률 범위: 0 ≤ P(A) ≤ 1
> 여사건을 활용하면 '적어도 하나' 유형이 간단해짐: P(적어도 하나) = 1 - P(하나도 없음)

### 시침·분침

**기본 이동 속도**

| 바늘 | 1시간에 이동 | 1분에 이동 |
|------|-------------|-----------|
| **분침** | 360° | 6° |
| **시침** | 30° | 0.5° |

**시각별 시침·분침 위치**

| 구하는 것 | 공식 |
|-----------|------|
| 분침의 각도 | 분 × 6° |
| 시침의 각도 | 시 × 30° + 분 × 0.5° |
| 두 바늘이 이루는 각도 | \|분침 각도 - 시침 각도\| (180° 초과 시 360°에서 뺌) |

**자주 나오는 유형**

| 유형 | 풀이 방법 |
|------|-----------|
| **두 바늘이 겹칠 때** | 분침 각도 = 시침 각도 → 분 × 6 = 시 × 30 + 분 × 0.5 |
| **두 바늘이 일직선(180°)일 때** | \|분침 각도 - 시침 각도\| = 180° |
| **두 바늘이 직각(90°)일 때** | \|분침 각도 - 시침 각도\| = 90° |

> 분침이 시침보다 **5.5°/분** 더 빠르게 이동 (6° - 0.5° = 5.5°)
> 두 바늘이 겹친 후 다시 겹칠 때까지: 360° ÷ 5.5° = 약 **65.45분**

### 수열 추리

| 종류 | 규칙 | 예시 | 주요 공식 |
|------|------|------|-----------|
| **등차수열** | 일정한 수를 더하거나 뺌 | 2, 5, 8, 11 … (+3) | n번째 항 = 첫째 항 + (n-1) × 공차 |
| **등비수열** | 일정한 수를 곱함 | 2, 6, 18, 54 … (×3) | n번째 항 = 첫째 항 × 공비^(n-1) |
| **계차수열** | 인접한 두 항의 차이가 규칙을 이룸 | 1, 2, 4, 7, 11 … (차이: 1,2,3,4) | 차이 수열을 먼저 파악 |
| **교차수열** | 홀수번째·짝수번째 항이 각각 다른 규칙 | 1, 3, 2, 6, 3, 9 … | 홀수·짝수 항 분리 후 각각 분석 |
| **피보나치형** | 앞 두 항의 합이 다음 항 | 1, 1, 2, 3, 5, 8 … | 앞 두 항 합산 |
| **군수열** | 규칙적으로 묶인 그룹이 반복 | (1), (2,3), (4,5,6) … | 그룹 단위로 묶어서 분석 |
| **반복수열** | 일정한 항의 묶음이 주기적으로 반복 | 1, 2, 3, 1, 2, 3, … | n번째 항: n ÷ 주기의 나머지로 위치 파악 |

> 풀이 순서: ① 차이 확인(등차) → ② 비율 확인(등비) → ③ 차이의 차이 확인(계차) → ④ 홀짝 분리(교차) → ⑤ 군수열/반복수열 확인
