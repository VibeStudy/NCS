## 문서이해능력

### 문서의 종류

| 문서 종류 | 설명 | 상황 | 
|-----------|------|----------|
| **공문서** | 행정기관에서 대내적 또는 대외적 공무를 집행하기 위해 작성하는 문서로, 보통 기관끼리 주고받는 문서 | 요청이나 확인 부탁 |
| **기안서** | 업무 협조를 구하거나 의견 전달의 '초안'에 작성하는 문서. '사내공문서' 또는 '회사 내 공문서' 라고도 불림. |
| **기획서** | 아이디어를 통해 도출된 프로젝트 시행을 상대방에게 설득하기 위해 작성하는 문서 <br>- 구성: 기획 목적 - 기획 내용 - 기대효과 | 제안이나 기획 필요 |
| **보고서** | 특정한 일에 관한 현황이나 진행상황, 연구/검토 결과 등을 보고하기 위해 작성하는 문서 |
| **설명서** | 상품의 특성이나 사물의 성질과 가치, 작동 방법이나 과정을 소비자에게 설명하기 위해 작성하는 문서 | 구체적인 정보제공 |
| **안내문** | 어떠한 행사나 사실(제도), 기업의 사안에 따른 안내 정보를 제공하는 문서 | 구체적인 정보 제공 | 
| **보도자료** | 정부기관이나 기업체, 각종 단체 등이 언론을 상대로 자신들의 정보를 기사로 보도되도록 하기 위해 작성하는 문서 |
| **품의서** | 업무의 수행과 관리를 시행하기에 앞서 결재권자에게 특정한 사안을 승인해 줄 것을 요청하기 위해 작성하는 문서 |
| **결의서** | 어떤 안건에 대해 수행을 목적으로 의사를 표시 및 결정한 것을 기록한 서식 |
| **자기소개서** | 기업이 요구하는 개인적인 배경이나 경력 등을 진술함으로써 입사, 계약 등의 목적을 위해 작성하는 문서 |
| **비즈니스 레터** | 사업상의 이유로 고객이나 단체에 공식적 또는 비공식적으로 편지를 쓰기 위해 작성하는 문서 |
| **비즈니스 메모** | 업무상 중요한 일이나 전화, 회의 등의 체크할 사항을 메모하기 위해 작성하는 문서 |


### 문서의 효력 발생 시기

| 원칙 | 설명 |
|------|------|
| **표백주의** | 의사표시가 성립한 때 효력 발생 |
| **발신주의** | 문서를 **발송(발신)한 때** 효력 발생 |
| **도달주의** | 문서가 상대방에게 **도달한 때** 효력 발생 (우리나라 공문서 원칙) |
| **요지주의** | 상대방이 문서 내용을 **실제로 인지한 때** 효력 발생 |

### 문서의 작성 방법과 기준

| 항목 | 기준 | 예시 |
|------|------|------|
| **용지** | A4(210mm × 297mm), 세로 방향 사용 |
| **여백** | 위 30mm / 아래 15mm / 왼쪽 20mm / 오른쪽 15mm |
| **글의 색채** | 검정색 또는 푸른색. 적색은 특별한 경우만 사용. |
| **제목** | 문서 내용을 압축한 간결한 표현으로 작성. 비유하지 않음 |
| **문서의 마지막** | - 문서의 마지막에는 '끝.' 사용.<br>- 첨부물이 있는 경우 첨부 표시 한 후 1자를 띄우고 '끝' |
| **숫자** | 아라비아 숫자 사용 | 3개, 15명, 100 |
| **날짜** | 연. 월. 일. 형식 (온점 + 띄어쓰기) | 2026. 2. 26. |
| **시간** | 24시간제, 시·분 사이 콜론 | 09:00, 14:30 |
| **금액** | 아라비아 숫자 + 괄호 안에 한글 병기 | 100,000원(십만 원) |
| **쪽 번호** | 문서 하단 가운데, 아라비아 숫자 | - 1 -, - 2 - |
| **바코드** | 문서 상단 행정기관명 표시줄의 오른쪽 끝에 2cm x 2cm 범위에 표시 |
| **보고자 표시** | 업무관리 시스템 또는 전자문서시스템을 이용하여 보고하거나 결재권자에게 직접 보고하지 않는 경우 생략함 |
| **띄어쓰기** | - 행정기관명은 붙여 씀 <br>- 직함과 직위는 띄어 씀 (xxx 과장)  <br>- 쌍점(:)은 시간을 제외하고는 앞말과 붙이고 뒷말과 띄어 씀(일시: 2026. 02. 26) |
| **항목 작성** | 단계별 기호 순서: 1. → 가. → 1) → 가) → (1) → (가) → ① → ㉮ |

## 경청능력

### 경청의 방해 요인

| 방해 요인 | 설명 |
|-----------|------|
| **짐작하기** | 상대방의 말을 끝까지 듣지 않고 미리 내용을 추측하여 결론 내림 |
| **대답 준비하기** | 상대방이 말하는 동안 자신이 할 말을 준비하느라 내용을 제대로 듣지 못함 |
| **걸러 듣기** | 자신이 듣고 싶은 말만 선택적으로 듣고 나머지는 무시함 |
| **판단하기** | 상대방에 대한 선입견이나 고정관념으로 내용을 왜곡하여 이해함 |
| **다른 생각하기** | 딴생각을 하거나 주의가 분산되어 집중하지 못함 |
| **조언하기** | 상대방의 말이 끝나기도 전에 해결책이나 충고를 제시하려 함 |
| **언쟁하기** | 상대방의 말에 반박하거나 논쟁하려는 태도로 경청을 방해함 |
| **자존심 세우기** | 자신에 대한 비판적 언급에 대해 강한 거부감을 드러내는 것 |
| **슬쩍 넘어가기** | 문제점이나 상대방의 부정적 감정을 회피하기 위해 유머를 사용하거나 초점을 빗겨가는 것 |
| **비위 맞추기** | 상대방의 기분을 맞추려고 동조만 하고 실제로는 집중하지 않음 |

### 효과적인 경청 방법

| 방법 | 설명 |
|------|------|
| **준비하기** | 강연이나 발표를 들을 때 주제를 미리 파악하고, 사전 지식을 쌓아두어 내용 이해를 높임 |
| **주의를 집중하기** | 말하는 사람의 얼굴, 몸짓 등을 주의 깊게 살피며 내용에 온전히 집중함 |
| **예측하기** | 상대방이 다음에 무슨 말을 할지 추측하면서 들으면 내용 파악에 도움이 됨 |
| **나와 관련짓기** | 들은 내용을 자신의 경험이나 지식과 연결하여 의미를 더 잘 이해함 |
| **질문하기** | 대화 중 이해가 안 되거나 더 알고 싶은 부분은 적절히 질문함 |
| **요약하기** | 들은 내용을 머릿속으로 정리하고 핵심을 파악하여 기억에 남김 |
| **반응하기** | 고개를 끄덕이거나 눈을 맞추는 등 적절한 비언어적 반응으로 경청하고 있음을 표현함 |

### 경청 훈련 방법

- 주의 기울이기
- 상대방의 경험을 인정하고 추가 정보 요청하기
- 정확성을 위해 요약하기
- 개방적인 질문하기
- '왜'라는 질문 삼가기

## 의사표현능력

### 효과적인 의사표현 방법

- 전달하고 싶은 의도, 생각, 감정이 무엇인지 분명히 인식해야 함
- 전달하고자 하는 내용을 명료하고, 적절한 메시지로 바꾸어야 함
- 메시지를 전달하려는 매체와 경로를 신중히 선택해야 함
- 듣는 이가 자신의 메시지를 어떻게 받아들였는지 피드백을 받아야 함
- 말하는 이의 표정, 음성적 특성, 몸짓 등 비언어적 방식을 활용해야 함
- 확실한 의사표현을 위해 반복적으로 전달해야 함

### 의사표현 기법

| 기법 | 설명 |
|------|------|
| **문간에 발 들여놓기 기법** | 작은 요청을 먼저 수락받은 뒤, 점차 큰 요청을 하는 기법. 일관성 심리를 이용 |
| **얼굴 부딪히기 기법** | 처음에 크고 무리한 요청을 하여 거절당한 후, 원래 원하던 작은 요청을 하는 기법. 상대방의 미안함을 이용 |
| **단순 노출 기법** | 같은 메시지나 사람을 반복적으로 노출시켜 친숙함과 호감을 높이는 기법 |
| **낮은 공 기법** | 처음에 유리한 조건을 제시해 동의를 얻은 후, 나중에 조건을 불리하게 바꾸는 기법. 이미 동의한 심리를 이용 |
| **엉덩방아 기법** | 유능한 사람이 실수(엉덩방아)를 했을 때 오히려 친근감과 호감이 높아지는 효과를 활용하는 기법 (= 프랫폴 효과) |
| **헤일로 기법** | 한 가지 긍정적인 인상(외모, 명성 등)이 다른 모든 특성까지 좋게 평가되도록 만드는 효과를 활용하는 기법 (= 후광 효과) |

#### 사이시옷

사이시옷은 합성어에서 뒷말 첫소리가 된소리로 나거나 'ㄴ' 소리가 덧날 때 표기

**표기 조건 (모두 충족해야 함)**
1. 합성어일 것
2. 앞말이 모음으로 끝날 것
3. 아래 중 하나:
   - 순우리말 + 순우리말
   - 순우리말 + 한자어
   - (예외) 한자어 + 한자어 6개: **곳간, 셋방, 숫자, 찻간, 툇간, 횟수**

| 구성 | 예시 |
|------|------|
| 순우리말 + 순우리말 | 냇가, 나뭇잎, 햇빛, 깻잎 |
| 순우리말 + 한자어 | 전셋집, 제삿날, 예삿일, 머릿수 |
| 한자어 + 한자어 (예외 6개) | 곳간, 셋방, 숫자, 찻간, 툇간, 횟수 |
| 한자어 + 한자어 (사이시옷 X) | 대가(代價), 초점(焦點), 고가(高架) |

### 어법 오류 유형

#### 주어-서술어 호응 오류

| 오류 문장 | 수정 |
|-----------|------|
| "제 꿈은 선생님이 되어 아이들을 가르치고 싶습니다" | "제 꿈은 선생님이 되어 아이들을 가르치는 것입니다" |
| "이 법안의 목적은 환경을 보호하기 위해서입니다" | "이 법안의 목적은 환경을 보호하는 것입니다" |

#### 이중 피동

피동 접미사(`-이/-히/-리/-기`)와 `-어지다`를 중복 사용하는 오류

| 오류 | 수정 |
|------|------|
| 잡혀지다 | 잡히다 |
| 쓰여지다 | 쓰이다 |
| 읽혀지다 | 읽히다 |
| 보여지다 | 보이다 |
| 만들어지다 | 만들어지다 (허용 — '-어지다'만 사용) |

#### 중복 표현

| 중복 표현 | 수정 | 이유 |
|-----------|------|------|
| 과반수 **이상** | 과반수 | '과반수' 자체가 '절반을 넘는 수' |
| **미리** 예약 | 예약 | '예약'에 '미리'의 뜻이 포함 |
| **역전** 앞 | 역전 / 역 앞 | '전(前)'이 이미 '앞' |
| **처음** 시작 | 시작 | '시작'이 이미 처음을 의미 |
| **뜨거운** 열기 | 열기 | '열(熱)'이 이미 뜨거움 |
| **모든** 전부 | 모든 / 전부 | 의미 중복 |
| **약** 10명 **정도** | 약 10명 / 10명 정도 | '약'과 '정도' 중복 |
| **피해를 입다** | 피해를 당하다 / 손해를 입다 | '피(被)'가 이미 '당하다' |

#### 번역투 표현

| 번역투 (오류) | 자연스러운 표현 |
|---------------|----------------|
| ~에 의해 (피동 남용) | 능동으로 전환 |
| ~의 경우에는 | ~는, ~이면 |
| ~을 가지다 | ~이다, ~하다 |
| ~에 대한 생각을 하다 | ~을 생각하다 |
| ~을 통해서 | ~로, ~으로 |

#### 호응 관계 (자주 출제)

| 앞 표현 | 맞는 서술어 |
|---------|------------|
| 결코, 절대로 | ~지 않다 (부정) |
| 아무리 | ~아도/어도 |
| 비록 | ~지만, ~더라도 |
| 만약, 만일 | ~(이)라면 |
| 왜냐하면 | ~기 때문이다 |

### 품사

| 품사 | 분류 | 설명 | 예시 |
|------|------|------|------|
| **명사** | 체언 | 사물의 이름 | 나무, 하늘, 서울 |
| **대명사** | 체언 | 명사를 대신하는 말 | 나, 너, 이것, 여기 |
| **수사** | 체언 | 수량·순서를 나타내는 말 | 하나, 둘, 첫째, 둘째 |
| **동사** | 용언 | 움직임·작용을 나타내는 말 | 가다, 먹다, 하다 |
| **형용사** | 용언 | 성질·상태를 나타내는 말 | 크다, 예쁘다, 좋다 |
| **관형사** | 수식언 | 체언을 수식하는 말 (활용 안 함) | 새, 헌, 이, 그, 저, 한 |
| **부사** | 수식언 | 용언·관형사·다른 부사를 수식하는 말 | 매우, 빨리, 다시 |
| **조사** | 관계언 | 체언 뒤에 붙어 관계를 표시하는 말 | 이/가, 을/를, 은/는, 에서 |
| **감탄사** | 독립언 | 감정·의지를 나타내는 독립적인 말 | 아, 오, 어머, 아이고 |

> 체언(명사·대명사·수사) / 용언(동사·형용사) / 수식언(관형사·부사) / 관계언(조사) / 독립언(감탄사)

**관형사 vs 수사 구별**

| 구분 | 예시 | 설명 |
|------|------|------|
| 관형사 | 한 사람, 두 마리 | 뒤에 명사가 옴 |
| 수사 | 하나, 둘, 셋 | 단독으로 수량·순서 나타냄 |

### 문장 성분

| 성분 | 분류 | 설명 | 예시 |
|------|------|------|------|
| **주어** | 주성분 | 동작·상태의 주체 (누가/무엇이) | **철수가** 간다 |

| **부사어** | 부속성분 | 용언·문장 전체를 수식 | **빨리** 가다 |
| **독립어** | 독립성분 | 문장과 독립된 성분 (감탄·호칭) | **철수야**, 밥 먹어라 |

> 주성분: 주어·서술어·목적어·보어
> 부속성분: 관형어·부사어
> 독립성분: 독립어

## 사자성어

| 사자성어 | 뜻 |
|----------|----|
| **이심전심** | 마음에서 마음으로 전함. 말하지 않아도 서로 통함 |
| **언중유골** | 말 속에 뼈가 있다. 예사로운 말 속에 날카로운 의도가 담겨 있음 |
| **구밀복검** | 입에는 꿀, 뱃속에는 칼. 겉으로는 친절하나 속은 음험함 |
| **일언반구** | 한 마디 말과 반 구절. 아주 짧은 말 |
| **유구무언** | 입은 있으나 말이 없다. 변명할 말이 없음 |
| **함구무언** | 입을 다물고 말이 없음 |
| **침소봉대** | 바늘을 몽둥이라 한다. 작은 것을 크게 과장하여 말함 |
| **과언불입** | 지나친 말은 받아들여지지 않음. 말은 삼가야 함 |
| **구화지문** | 입은 재앙이 드나드는 문. 말을 함부로 하면 화를 부름 |
| **일구이언** | 한 입으로 두 말을 함. 말을 이랬다저랬다 바꿈 |
| **감언이설** | 달콤한 말과 이로운 말. 남의 비위를 맞추는 교묘한 말 |
| **동문서답** | 동쪽을 묻는데 서쪽을 답함. 질문과 전혀 다른 엉뚱한 답변 |
| **횡설수설** | 조리 없이 이리저리 마구 떠들어 댐 |
| **부화뇌동** | 자신의 소신 없이 남의 의견에 무조건 따름 |
| **아전인수** | 내 논에 물 대기. 자기에게 유리하게 해석하거나 주장함 |
| **역지사지** | 처지를 바꾸어 생각함. 상대방 입장에서 이해하려는 태도 |
| **백문불여일견** | 백 번 듣는 것이 한 번 보는 것만 못함 |
| **언행일치** | 말과 행동이 일치함 |
| **다언삭궁** | 말이 많으면 자주 궁지에 몰림. 말을 아껴야 함 |
| **일언지하** | 한마디로 딱 잘라 말함 |
