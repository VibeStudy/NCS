# 수리능력 풀이 팁

## 의자에 나눠 앉는 문제

**방정식 vs 부등식 판단 기준: 마지막 의자의 인원수가 주어졌는지 여부**

| 조건 | 유형 | 이유 |
|------|------|------|
| 마지막 의자에 정확히 몇 명 앉았는지 명시 | **방정식** | 경우의 수가 하나로 확정됨 |
| 한 의자당 몇 명씩 앉는다고만 명시 | **부등식** | 마지막 의자는 1명~정원까지 가능, **최대/최소**를 물어봄 |

**예시**

> `의자에 6명씩 앉으면 마지막 의자에는 2명이 앉고, 의자 2개가 남았다.`
> → **방정식** (마지막 의자 인원 2명으로 확정)

> `의자에 8명씩 앉으면 의자 2개가 남았다.`
> → **부등식** (마지막 의자 인원 불명확, 1~8명 가능)

## 케이스 나열 방법

케이스를 빠짐없이 정리할 때는 **끝자리부터 1씩 증가**시키며 나열한다.

> 중구난방으로 쓰면 중복·누락이 생기기 쉬움

**예시: 두 자리, 각 자리 0~3**

```
0 0
0 1
0 2
0 3
1 0
1 1
1 2
1 3
2 0
...
```

> 앞자리는 뒷자리가 끝까지 다 돌았을 때만 올림 → 숫자 세는 것처럼 순서대로

## 원형 테이블 문제

n명을 원형으로 배열하는 경우의 수 = **(n-1)!**

> 원형은 회전해도 같은 배열로 보기 때문에 한 명을 고정하고 나머지 (n-1)명을 배열
> 예) 6명을 원형 테이블에 앉히는 경우의 수 = 5! = 120
